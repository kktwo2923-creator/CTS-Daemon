导入收款账单 → 订单号自动发卡

导出微信/支付宝「二维码收款」账单(.xlsx)后选择文件即可导入。之后买家在购买页填的付款单号若与账单里「收入」记录匹配且金额达标,系统自动发卡,无需手动确认。可重复导入(已存在的自动跳过)。

订单 / 确认到账发卡

买家付款后在购买页填单号提交,这里出现「待确认」。核对到账后点「确认发卡」或「一键通过」,买家即可用单号取卡。
